How to install browser sync with sublime text 3


Steps:

1. Hit ctrl + shift + p (for windows, command + shift + p for apple) to bring up the Command Pallate.

2. Start typing Install and you should see a pop up that says Package Control: install package. Install it.

3. You should see something called Browser Packages tab on top of sublime 3. if not, you have to go to the website linked bellow and follow those instructions to download it.

BrowserSyncDownloadHelp

4. Once you have downloaded Broswer Sync and see the tab on top of sublime, click it, then click launch. If you use chrome, you are good to go and it works! Otherwise, follow these last set of steps.


Steps if you don't use chrome:

1. Navigate to the Preferences tab, located next to the Help Tab.

2. Click Browse Package.s

3. It will open up a folder with 2 folders inside, one should say user and one should say BrowserSync.. Click on Browser Sync.

4. Look for the application called browser_sync_launch. You will DRAG AND DROP INTO SUBLIME TEXT, DON'T DOUBLE CLICK IT. It should open a new file in sublime text.

5. Lastly, go to where it says browser:"chrome" and change chrome to whatever browser you will be using for HTML,CSS,and Javascript coding!!

6. To launch, do the same thing as Step 4 in the last part. Have fun coding!!
